# Diary Planner_toy project



### 프로젝트 개요

2023.08.22 시작한 토이 프로젝트

일기장 기능과 매일 작성하는 플래너 기능을 제공

------



### 주요기능

#### 회원

- 회원 가입 및 이메일 인증



#### 일기장

- 일기장 작성 기능 
- 매일 다음날 02:00 이후로는 작성 불가



#### 플래너 기능

- 매일 작성할 수 있음 
- 카테고리를 생성하고 달 수 있도록 할 것

------
